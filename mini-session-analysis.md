## 2. クライアントのタイプ分類

### タイプA：不倫・浮気被害型
夫の不倫・浮気が発覚し、証拠収集・再構築・離婚検討の渦中にある。

**該当：ゆりこさん・ひとみさん・のぞみさん・朋子さん・ももこさん**

共通の特徴：
- 裏切られた事実と向き合いながらも、夫への愛情や依存が残っている
- 「なぜ私じゃダメだったのか」という自己否定に陥りやすい
- 証拠収集・夫の監視など、コントロール行動でなんとかしようとする
- 怒りの裏に「見捨てられたくない」「もう一度愛されたい」がある

### タイプB：感情抑圧・我慢蓄積型
怒りや不満を言葉にせず飲み込んでいるうちに、心身の限界に達している。

**該当：えりさん・みどりさん・ゆきさん・幸代さん・かすみさん・桂さん**

共通の特徴：
- 「言っても変わらない」「言ったら関係が壊れる」という諦め
- 「私ばっかり」という消耗感・報われなさ
- 我慢していることに気づいていない（我慢が「普通」になっている）
- 夫ではなく「自分が悪いのかも」と思いやすい

### タイプC：セミナージプシー型
複数の学びを重ねてきたが、現実に変化が出ていない。

**該当：典子さん・ゆみこさん・恵さん・侑里さん**

共通の特徴：
- インプットが多いが腑に落ちていない（頭でわかっているが感情が動いていない）
- 「次の何かで変われるはず」という外部依存
- 根底の「自分には価値がない」「このままでは愛されない」が変わっていない
- 知識で自分を防衛している側面がある

### タイプD：育休孤立型
育休中に夫との溝が一気に深まり、孤独と怒りが爆発しそうになっている。

**該当：亜利沙さん・のぞみさん・ゆりこさん**

共通の特徴：
- 社会から切り離された孤立感
- 子育ての責任を一手に担っている疲弊
- 夫が「仕事」を盾にして家庭に関わらない構造への怒り
- 産後の心身の変化の中で、自分を見失っている

### タイプE：複合課題・生活基盤型
夫婦問題に加えて、家計・仕事・育ち・ネットワークなど外部課題も複合している。

**該当：えみさん・しおりさん・ゆきさん**

共通の特徴：
- 夫婦間の感情的問題の前に、生活レベルの不安や不信がある
- 「この人と一緒にいて大丈夫か」という根本的な問いがある
- 夫への愛情よりも、現実の不安が先に立っている

## 3. 彼女たちが持っている「前提・常識・信じていること」

### 前提①：「妻・母である私が、我慢して当たり前」
- 家庭を守るのは女性の役割という刷り込み
- 我慢が「愛情の証明」になっている
- 自分の感情を出すことへの罪悪感

### 前提②：「夫を変えれば、問題が解決する」
- 「夫が変われば私は幸せになれる」という外部依存
- 自分が変わることへの無自覚または抵抗
- 相手をコントロールしようとするアプローチに陥る

### 前提③：「感情を出してはいけない」
- 怒ったり泣いたりすることは「弱さ」または「迷惑」
- 穏やかでいることが「良い妻・良い母」の条件
- 感情を抑えることでしか、関係を保てないと思っている

### 前提④：「正しいことを言えば、相手はわかってくれるはず」
- 論理で説明すれば理解されるという期待
- 感情のコミュニケーションより「正しさ」を重視
- 言っても変わらないと、さらに絶望が深まる

### 前提⑤：「自分の価値は、誰かの評価で決まる」
- 夫に認められないこと＝自分に価値がないこと
- 承認されることで初めて、自分の存在が安心できる
- 「私って、大切にされていいの？」という根本の問い

### 前提⑥：「学べば・知れば、変われる（はず）」（セミナージプシー型に顕著）
- 知識の蓄積が変化につながると信じている
- 感情レベルの変化より、頭での理解を優先
- 「次の何かで変われる」という期待が続く

## 4. なぜ夫婦仲を拗らせているのか（深層心理の分析）

### 感情の3層構造

```
表層：怒り・不満・批判（「なんでわかってくれないの」）
　　　　　　↓
中層：虚しさ・疲れ・孤独（「もう頑張れない」「報われない」）
　　　　　　↓
深層：寂しさ・恐怖・愛されたい（「私って愛されてる？」「見捨てられたくない」）
```

多くのクライアントが表層の怒りや不満で夫と関わっているため、深層にある「寂しさ・愛されたい」が伝わらず、すれ違いが続く。

### 根底の構造：「自己価値の外注」

**自分の価値を、自分で決められていない。**

- 夫に大切にされる → 自分には価値がある
- 夫に無視される → 自分には価値がない

このパターンが続くと：
1. 夫の言動に過剰反応する（自分の価値が脅かされるから）
2. 夫をコントロールしようとする（自分の価値を守るために）
3. 夫が変わらないと絶望する（自分の価値が回復されないから）
4. さらに関係が悪化する

### 5つの拗らせパターン

**パターン①：我慢の蓄積爆発型**
日常的な我慢 → ある日限界突破 → 怒りの爆発 → 夫の撤退 → 孤立感の増大

**パターン②：コントロール失敗型**
夫を変えようとする → 夫が反発・距離を置く → さらに強くアプローチ → 関係の硬直化

**パターン③：自己否定ループ型**
夫に否定される → 「やっぱり私が悪いのかも」→ 自分を責める → 自信喪失 → さらに承認を求める

**パターン④：被害者ポジション固定型**
「私はこんなに頑張っているのに」という視点が固定 → 夫を加害者として見続ける → 関係改善の糸口が見えない

**パターン⑤：頭でわかっているけどできないループ型**（セミナージプシー型）
知識を得る → 頭では理解 → 行動・感情が変わらない → 「もっと学べば変われる」→ 次のセミナーへ

## 5. Coa-Terasuのアプローチが効く理由

上記の分析から、クライアントに本当に必要なのは：

1. **「夫を変える」ではなく「自分との関係を変える」という視点転換**
   - 外部依存から、自己起点へのシフト

2. **感情の3層を丁寧に降りていくプロセス**
   - 怒りの下にある「寂しさ」に気づくこと
   - その寂しさを、夫以外の方法で満たせるようになること

3. **「私が私の一番の味方」になる体験**
   - 承認を外に求めない内側の安心感
   - 自分の価値を自分で決めるプロセス

4. **「わたし基準」で選び直すこと**
   - 誰かのための人生から、自分の人生へ
   - 我慢ではなく、選択として生きる

- 「怒りの下に、ずっと泣いている私がいた」
- 「承認がないと不安なのは、自分を信じていなかったから」
- 「セミナーに行っても変われなかったのは、感情を扱っていなかったから」

### ターゲット層が持っている言葉（そのまま使える）
- 「なんで私ばっかり」
- 「言っても変わらない」
- 「もう疲れた」
- 「夫のことが嫌いじゃないけど、一緒にいるのがしんどい」
- 「自分が悪いのかとも思う」
- 「感情的になってしまう自分が嫌」

### サービス設計への示唆
- **初期段階**：感情の3層構造への気づきが最重要（特にミニセッション・無料コンテンツで）
- **中期段階**：「自己価値の外注」パターンを自分で気づき、手放すプロセス
- **後期段階**：「わたし基準」で日常の選択ができるようになる実践
