# ビフォー指標3問 — 初回セッション運用シート

> ちほさん用（クライアントには「指標」と言わず、自然な会話で聞く）。
> 根拠: みきさんの「立ち直り半日→5分」は偶然本人が語ったから残った。変化を自分で認められない方ほど、初回の数字が後で効く（統合分析・提案3）。

---

## 初回に聞く3問（本人の言葉のまま記録する）

1. **「嫌なことがあった時、立ち直るのに今、どれくらいかかりますか？」**
   （例: 半日引きずる／寝るまで／2〜3日）
2. **「夫さんに本音を言えたことって、最近ひと月でどれくらいありますか？」**
   （例: ゼロ／1回だけ／言いかけてやめた）
3. **「自分のためだけの時間って、週にどれくらい取れていますか？」**
   （例: ほぼゼロ／子どもが寝た後の30分／覚えていない）

**聞き方のコツ**: 尋問にしない。悩みの棚卸しの流れで「ちなみに、今って…」と自然に。数字が出なければ本人の表現のまま（「ずーっと引きずる」等）でOK。**正確さより、本人の言葉であることが大事。**

## 記録先

各クライアントの `変化の記録.md` の冒頭に以下の表を追加する（振り返りシートには載せない。ちほさん側の記録）。

```
## ビフォー指標
| 指標 | 初回 | 6回目 | 12回目 |
|------|------|-------|--------|
| 立ち直りにかかる時間 | 「」 | | |
| 本音を言えた回数/月 | 「」 | | |
| 自分のための時間/週 | 「」 | | |
```

## 再測定のタイミングと使い方

- **6回目前後**（揺り戻しの時期）: 「全然変わっていない」と本人が言った時にこそ、初回の言葉を見せる。「初回は『半日引きずる』とおっしゃっていましたよ」——揺り戻し期の一番の証拠になる
- **12回目**: ビフォーアフターとして本人に返す。継続のご提案の根拠にも、（同意が取れれば）お客様の声にもなる
- **注意**: 数字が伸びていない指標があっても、詰めない。「ここはこれからの伸び代ですね」で扱う（変化は直線でない）

## 仕組みへの組み込み（設定済み）

- 新規クライアントの初回前準備メモ（振り返りシートスキル・モードA）に、この3問を含める
- 6回目・12回目の準備メモに「再測定」のリマインドを含める
