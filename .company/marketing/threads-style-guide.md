> ブランドトーン・NGワードは `CLAUDE.md` を参照。

## 発信の基本方針

| 項目 | 内容 |
|------|------|
| 読んだ人に持ってほしい印象 | 信頼できる専門家 |
| 主要プラットフォーム | Threads |
| 文体 | です・ます調（体言止め・短い断言も積極的に使う） |
| 絵文字 | 使わない |
| ハッシュタグ | 使わない |
| 文字数の目安 | スレッド①：60〜100文字／スレッド②③：500〜800文字 |

---

## 声のトーン（ちほさんの分身として）

- **凛としていて、温かい**：押しつけず、ただそこにいる。静かに寄り添う専門家
- **断言するが、裁かない**：「〜です」と言い切りながら、誰も責めない
- **自分ごととして書く**：「〜な人は〜しがち」という外部分析は絶対にしない。読者の体験をそのまま描写する
- **余白を作る**：「〜かもしれません」「〜てきたかもしれません」で読者に受け取る余地を残す

---

## 文体の特徴

- **です・ます調を基本**にしつつ、体言止めや短い断言文を積極的に織り交ぜる
  - 例：「正論では、人を動かすことはできません。」「でも、あなたはゴミ箱じゃない。」
- **逆説・否定から入る**：「〜ではありません」「〜することでもありません」で読者の思い込みを静かに崩す
- **感情はラベルでなく言葉のまま置く**：「不安」ではなく「寂しかった。わかってほしかった。」
- **箇条書きは「・」**：具体例を3つ程度並べる。前後に空行を入れる

---

## 改行・空白のルール

- 基本は**1〜2文で改行**。長い文は途中でも改行する
- 感情や核心的な一文は**1行単独**に置く
  - 例：「ただ、大切にされたい。安心したい。つながりを感じたいだけ。」
- 箇条書きの前後は**空行**を入れる
- 意味のまとまりの区切りに空行を入れ、読む呼吸を作る

---

## 投稿順のルール

- 同じテーマは**3投稿以上空けてから**繰り返す
  - 例：A → B → D → A はOK　／　A → B → A はNG
- Aテーマ（逆説・否定形）はリーチが高いが、連続させると飽きられる。間隔を意識して使う

## Threads投稿の構成（スレッド分割型）

### スレッド①：冒頭ティーザー（答えを明かさず続きへ）

**型：**
```
〔テーマ〕な妻が〇〇していること。
それは、〜でも
〜でもありません。
〔動詞句〕のは、
```

- スレッド①の締め言葉は `しないのは、` `手放したのは、` `やめたのは、` `一番大きいのは、` など、テーマに合った動詞句で終わる
- `実は、` は使わない
- 答えはスレッド②の冒頭で明かす
- **淡々とした事実の提示**から始める。問いかけ禁止
- 「〜ではありませんか？」は絶対に使わない

**実例：**
```
夫婦仲がいい妻が、絶対にしないこと。
それは、感情的にならないことでも、
不満を言わないことでもありません。
しないのは、
```

---

### スレッド②③：本編（心の仕組みを紐解く5ステップ）

**スレッド②は必ず「答え」から始める**

スレッド①の動詞句（`しないのは、` など）を受けて、カギカッコで答えを1行〜2行で置く。前後に空行を入れて際立たせる。

```
「夫の機嫌を、
今日の自分の価値に変換すること」
```

この答えが最も読まれる一文になる。短く、体言止めで、日常の言葉で書く。

**Step 1｜行動を淡々と描写する**
- 読者の「昨日・今朝」にあった場面を、感情を煽らずそのまま置く
- 「〜かもしれません」で読者に受け取る余地を残す

```
（例）
機嫌が悪そうだと、
「何かしたかな」と緊張する。
「今は黙っておこう」と口をつぐむ。

そうやって、自分の気持ちよりも
相手の顔色を
優先してきたかもしれません。
```

**Step 2｜心の癖・無意識の前提として名付ける**
- 行動を責めず、心の構造・前提として示す
- 「でもそれは、〜ではなく〜という心の癖です。」
- または「それは、〜という前提が無意識に働いているサインです。」

```
（例）
でもそれは、優しさではなく
相手の感情まで
自分が背負ってしまう心の癖です。
```

**Step 3｜「でもそれは〜ではありません」で責任を外す**
- 読者が自分を責めているところから解放する
- 「努力が足りないからでも、優しさが不足しているからでもありません。」

```
（例）
でもそれは、
あなたの我慢が足りないからでも
優しさが不足しているからでもありません。
```

**Step 4｜本当の原因を1行で断言する**
- 短く、体言止めか「〜だけ。」で締める

```
（例）
自分を後回しにする関わり方を
続けてきただけ。
```

**Step 5｜「夫婦仲がいい妻は〜」で対比する**
- 理想を押しつけず、対比として静かに提示する

```
（例）
夫婦仲がいい妻は、
夫の感情を背負いません。
自分の感情を大切に扱います。
```

## E. 教育・投資型の投稿構成（特別ルール）

> **目的**: リーチより属性の絞り込み。「自分でなんとかなる」と思っている層に静かに揺さぶりをかけ、自己投資を覚悟できる人だけを個別相談に呼ぶ。いいね数・リーチ数は指標にしない。

### Eテーマの10サブパターン（コンサル資料より）

| # | サブタイプ | 核心メッセージ |
|---|-----------|--------------|
| E-1 | 自己変革マインドセット型 | 人生を変えた妻が最初に決めたのは「投資する」こと |
| E-2 | 先延ばし損失の可視化型 | 「いつか」を選ぶたびに「決めない自分」が強化される |
| E-3 | 自己投資ブロック破壊型 | 自分へのお金を贅沢と感じる思い込みを崩す |
| E-4 | 自力救済幻想の打破型 | 「自分でなんとかなる」は、まだ余白がある証拠 |
| E-5 | 体験談型（ちほさん自身） | 100冊読んでも変わらなかった。限界を知って初めて決断できた |
| E-6 | 決断の筋肉型 | 差は才能でも環境でもなく「決めた回数」 |
| E-7 | 自己投資罪悪感型 | 枯れたままで家族を満たすことはできない |
| E-8 | お金がないを問う型 | 本当にお金がないのではなく「自分に使う価値がない」と感じている |
| E-9 | 先送りコスト型 | 「もう少し落ち着いたら」を選び続けて5年経った人を見てきた |
| E-10 | 決めた瞬間に変わる型 | 受けると決めた、その瞬間から始まる |

> **ちほさんへのメモ**: E-4・E-2・E-6が「自分でなんとかなると思っている人を揺さぶる」狙い目。E-5（体験談型）はちほさんにしか書けない強み。まず3〜4本絞って自分の言葉でリライトする。

### Eテーマの文章原則

**「説明」ではなく「感情の因果関係」で流れる文章を書く。**

各パートが論理的につながるだけでなく、感情の連鎖として一本の糸になっていること。

```
例：「自分にお金を使えない」
　　　↓
　　「なぜ止まるのか」
　　　↓
　　「その結果、何が起きるのか」
　　　↓
　　「だから前提を変える」
```

**ターゲット層のリアルな声をそのまま置く**

以下のような言葉は、読者の内側にある実際の声。投稿の中にそのまま使う。

- 「私さえ我慢すれば」
- 「私なんかに…」
- 「もう少し落ち着いたら」
- 「自分でなんとかなる」
- 「こんな金額、私に使っていいの？」

**着地は「覚悟・選択・行動」まで持っていく**

気づきで終わらせない。「内省して気づく」だけでなく、「だから私はこれを選ぶ」という能動的な一歩まで見せる。

---

### Eテーマの投稿構成（スレッド分割型）

**スレッド①：冒頭ティーザー**

A〜Dテーマと同じ逆説・否定形テンプレートを使う。

```
夫婦関係を本気で立て直した妻が、一番最初に手放したもの。
それは、夫への期待でも、
我慢の癖でもありません。
手放したのは、
```

**スレッド②：本編**

Eテーマはステップ構成よりも「感情の因果関係」の流れで書く。

1. **場面・事実を置く**：読者の「昨日・今朝」にあった具体的な場面とセリフ
2. **止まる理由を名付ける**：行動を責めず、その奥にある感覚・前提を一言で言語化
3. **結果を見せる**：このままでいると何が起きるか。「愛情が我慢に変わる」など感情的な変化で示す
4. **着地・再定義**：「〜は贅沢ではなく、〜だ」という言い換えで読者の前提を静かに崩す
5. **CTA**：パターンA（ガイドブック誘導）またはパターンB（フォロー誘導）どちらも可。投稿内容とガイドブックが直結する場合はAを使う。リーチ数・いいね数は指標にしない

**Eテーマ専用のNGルール**

- 「お金がないから無理」という人を責める・見下す表現は使わない
- 「行動できない人はダメ」という二項対立にしない
- 投資額・分割払いの具体的な金額には触れない
- 「高額投資をすべき」という指示にしない。「選択の意味を問い直す」で留める
- 「救う」「魔法のような」表現は使わない（ブランドルール共通）

## CTAへのブリッジ構造

### パターンA：ガイドブック誘導（基本形）

```
とはいえ、

長年の〜は、
そう簡単には抜けませんよね。

〔解決策の一言〕を
ガイドブックにまとめました。
〔対象者〕は、プロフィールから受け取ってください。
→
```

**ルール：**
- `とはいえ、` はひらがな表記で統一（`とは言え、` は使わない）
- `とはいえ、` の後は必ず空行
- 「長年の癖」「長年の前提」への共感を必ず入れる
- 「ガイドブック」という言葉を使う
- 最後は「→」で締める
- 投稿内に「LINE」「ミニセッション」の言葉は出さない

**実例：**
```
とはいえ、
長年染みついた反応は深い。
気づくだけでは、
抜け出せないこともある。

なぜ繰り返すのか。
どこから整えるのか。
その構造と手順を、
ガイドブックにまとめました。

本気で変えたい方だけ、
プロフィールから受け取ってください。
→
```

---

### パターンB：フォロー誘導（ガイドブックを出さない投稿に）

ガイドブックへの誘導ではなく、フォロワー獲得を目的とする場合に使う。

```
〔自分を大切にする選択〕をこれからの人生に増やしていきたい人は、
フォローして〔日常のヒント〕を受け取ってくださいね。
→
```

**実例：**
```
自分を大切にする選択を
これからの人生に増やしていきたい人は、
フォローして日常を整えるヒントを
受け取ってくださいね。
→
```

## NG表現チェックリスト

投稿前に確認する。

- [ ] 冒頭が「〜ではありませんか？」の問いかけになっていない
- [ ] 「〇〇な人は〜しがち」という外部分析になっていない
- [ ] 「夫が悪い」「相手が変われば」という構図になっていない
- [ ] 評価語（ダメ・良くない・すべき）を使っていない
- [ ] 専門用語（投影、メタ認知、外在化）をそのまま使っていない
- [ ] 絵文字を使っていない
- [ ] ハッシュタグをつけていない
- [ ] 「本気の方だけ」「〇〇できない人は」という選別をしていない
- [ ] CTAで「LINE」「ミニセッション」という言葉を使っていない
- [ ] 「〜してください」という指示でCTAを終えていない
